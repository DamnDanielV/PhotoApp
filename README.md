# Fullstack App

## Descripcion

### Aplicativo web que administra un album fotografico con sus respectivos autores, manejando un CRUD con NodeJs a una base de datos relacional en Mysql y generando un servicio restApi del cual se consumiran los datos desde el frontend usando Angular 

## Tecnologias usadas

- FrontEnd: Angular
- Backend: NodeJs
- Database: Mysql
- RestAPI

- typescript

### Siguientes etapas:

- Relacionar con MongoDb
- Implementar autenticacion, Auth0 o JWT (Json Web Tokens)
- Optimizar el componente de busqueda por titulo.
